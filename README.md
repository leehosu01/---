# Election-rate-based-on-vote-counting
-------------

대한민국의 대통령 선거의 지역별 실시간 개표상황과 총 선거인수를 바탕으로 1분마다 득표수 추정을 수집합니다.

## usage
--------

```shell
python3 main.py
```
